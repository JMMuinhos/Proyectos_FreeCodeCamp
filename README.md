# Celestial Database

Proyecto de freeCodeCamp: base de datos **`universe`** sobre galaxias, estrellas, planetas, lunas y observatorios.

---

## Tablas

- **galaxy**: galaxias con tipo, edad y distancia.
- **star**: estrellas con tipo, temperatura, masa y referencia a su galaxia.
- **planet**: planetas con radio, distancia orbital, vida y referencia a su estrella.
- **moon**: lunas con tamaño, forma y referencia a su planeta.
- **observatory**: observatorios con año de fundación, telescopios y presupuesto.

---

## Características

- Primary keys autoincrementales.
- Relaciones definidas con foreign keys.
- Uso de **INT, NUMERIC, TEXT, VARCHAR y BOOLEAN**.
- Datos de ejemplo incluidos para cumplir los tests de freeCodeCamp.
